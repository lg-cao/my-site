这是广东科学技术职业学院商学院商务数据分析与应用专业操龙国老师的个人主页。

## 项目结构

```
├── index.html      # 主页内容
├── style.css       # 样式文件
├── script.js       # 交互功能
└── README.md       # 项目说明
```

## 技术栈

- HTML5
- CSS3
- JavaScript
- Font Awesome 图标库

- 响应式设计，适配不同设备
- 平滑滚动导航

## 部署指南

### 步骤1：创建GitHub仓库

1. 登录GitHub账号
2. 创建一个新的仓库，命名为 `caolongguo.github.io`（如果使用GitHub Pages）或任意名称
3. 将本地项目推送到GitHub仓库

### 步骤2：使用Cloudflare Pages部署

1. 登录Cloudflare账号
2. 导航到Cloudflare Pages
3. 点击 "Create a project"
4. 选择连接到GitHub仓库
5. 选择刚刚创建的仓库
6. 配置构建设置：
   - Framework: 选择 "None"
   - Build command: 留空
   - Build output directory: 留空（默认为根目录）
7. 点击 "Deploy site"
8. 部署完成后，Cloudflare会提供一个域名访问你的个人主页

## 自定义修改

1. **修改个人信息**：编辑 `index.html` 文件中的个人信息、研究方向、教学工作和学术成果等内容
2. **修改样式**：编辑 `style.css` 文件调整颜色、布局等样式
3. **修改交互**：编辑 `script.js` 文件调整交互功能
